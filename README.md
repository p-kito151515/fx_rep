# BTC水平線ドロワー

BTCチャートに水平線（サポート・レジスタンスゾーン）を自動描画するWebアプリ。  
Claudeプロジェクトと連携して使用する。

-----

## ファイル構成

```
fx_rep/
├── index.html          # メインのWebアプリ（これをGitHub Pagesで公開）
├── project_knowledge.md # Claudeプロジェクトの知識ファイル（プロジェクトに登録する）
└── README.md
```

-----

## 使い方

### セットアップ

1. このリポジトリをパブリックにする
1. GitHub Pages を有効にする（Settings → Pages → main ブランチ）
1. `project_knowledge.md` をClaudeプロジェクトの「プロジェクトの知識」に登録

### 毎回の使い方

**① Claudeプロジェクトでチャートを分析**

- チャートのスクリーンショットをプロジェクトのチャットに貼る
- Claudeが以下フォーマットで水平線データを出力する

```
上端: 74707
下端: 69800

73238,73490 | 5.0 | 最重要レジスタンス | #FF4444
73767 | 2.5 | 上レジスタンス | #FF8C42
71416 | 3.5 | 中間ピボット | #FFD700
70968 | 3.0 | 現在値サポート | #4CAF50
70075 | 4.5 | 強サポート | #00BCD4
69500 | 3.5 | 深めのサポート | #9C27B0
```

**② ドロワーで描画**

- GitHub Pages の URL を開く（ホーム画面に追加推奨）
- スクリーンショットを写真ライブラリから選択
- ①のテキストをそのまま貼り付け
- 「水平線を描画する」を押す
- 「📥 画像をダウンロード」で保存

-----

## 水平線フォーマット

```
価格 | 強度(2.5〜5.0) | 名前 | 色(hex)
```

- ゾーン指定: `73238,73490 | 5.0 | レジスタンス | #FF4444`
- 単線指定: `70075 | 4.5 | 強サポート | #00BCD4`
- 色は省略可（自動割当）

### 強度の基準

|強度 |意味                |用途     |
|---|------------------|-------|
|5.0|D1/H4/H1で重なる最強帯   |逆張り最優先 |
|4.5|H4で強く機能、ラウンドナンバー近く|逆張り候補  |
|4.0|H4で機能、複数回タッチ      |条件付き逆張り|
|3.5|H1で機能、上位足でも意識     |反応確認後  |
|3.0|H4で反応候補、単独では弱い    |利確目安   |
|2.5|H1のみ、ヒゲ中心         |通過点    |

## GitHub Pagesで公開する手順

1. リポジトリを **Public** にする（Settings → Danger Zone → Change visibility）
1. Settings → Pages → Source: `main` ブランチ → Save
1. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]/` でアクセス可能

-----

## Claudeプロジェクトの設定

`project_knowledge.md` をプロジェクトの知識に登録すると：

- チャート画像を貼るだけで水平線データが自動出力される
- 「ドロワーを出して」でHTMLアーティファクトが表示される
